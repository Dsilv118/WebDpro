13. 페이징 게시판

● 페이징 게시판
게시판 글을 페이징처리해서 글 당 번호, 한 페이지의 시작 번호, 한 페이지의 끝 번호를 지정한다.
글 아래 번호를 이용해 글이 많아졌을 때 넘어갈 수 있도록 처리한다. 

◎ 페이징 공식
PAGESIZE    : 한 페이지당 글 수 
BLOCKSIZE   : 블록당 페이지 수
currentPage : 현재 몇 페이지인지 (int)
pageNum    : 현재 몇 페이지인지 (String)

startRow     : 시작번호 
	ㄴ>(현제 페이지 - 1) * 페이지당 갯수 + 1                         

endRow      : 끝번호
	ㄴ>시작번호 + 페이지당 갯수 - 1

pageCnt      : 페이지 수
	ㄴ>Math.ceil((double)전체글 갯수/페이지당 갯수)

startPage     : 시작 페이지
	ㄴ>((현제페이지 - 1)/블록당 페이지 수) * 블록 당 페이지 수 + 1

endPage      : 끝 페이지
	ㄴ>(시작페이지 + 블록당 페이지 수 - 1)
