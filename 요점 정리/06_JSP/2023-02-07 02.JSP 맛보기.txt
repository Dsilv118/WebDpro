02. JSP 맛보기

● JSP (Java Server Pages)
JSP란 HTML 코드에 JAVA 코드를 넣어 동적 웹 페이지를 생성하는 웹 어플리케이션 도구이다.

◎ JSP 특징
* Java EE 스펙 중 하나로 웹 애플리케이션 서버에서 동작함
* JSP는 실행시에 자바 서블릿으로 변환되어 실행되어 서블릿과 거의 유사함. 그러나 서블릿과 달리
  HTML 표준에 따라 작성되므로 웹 디자인에 편리함
* 아파치 스트럿츠나 자카르타 프로젝트의 JSTL 등의 JSP 태그 라이브러리를 사용하는 경우 
  자바 코딩 없이 태그만으로 간략한 기술이 가능하므로 생산성을 높일 수 있음.

◎ JSP 동작 과정
① 브라우저가 웹 서버에게 JSP에 대한 요청 정보를 전달한다.

② 브라우저가 요청한 JSP가 최초로 요청했을 경우만 JSP로 작성된 코드가 서블릿 코드로 변환된다.

③ 서블릿 코드를 컴파일해서 실행가능한 bytecode로 변환한다.

④ 서블릿이 실행되어 요청을 처리하고 응답 정보를 생성한다.
