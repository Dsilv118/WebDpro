02. jQuery 기본 문법

● jQuery 기본 문법

◎ 선택자 
함수가 적용되길 원하는 요소 및 속성
* 아이디 선택자 : $("#id명");
* 태그 선택자    : $("태그명");
* 클래스 선택자 : $(".ID명");
* 속성 선택자    : $("태그명[속성명=속성값]");

◎ 값 가져오기
* $("선택자").val( );
* let inputTag = $("#text");
* inputTag.val( );

◎ 문자열 변경
* $("선택자").text( ); : 해당 태그에 쓰여진 문자열
* $("선택자").text("새로운 문자열"); : 태그에 쓰여진 기존 문자열 변경

◎ 스타일 변경
* $("선택자").css("속성명", "속성값");

◎ 반복문
* $.each(배열명, function(index, item) {
	index : 0부터
	item : 요소
  };
