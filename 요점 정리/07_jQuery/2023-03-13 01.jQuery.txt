01. jQuery

● jQuery
jQuery란?
자바 스크립트 라이브러리 중 가장 인기있는 라이브러리이다. 이벤트 대응, 애니메이션과 메뉴 효과등에서
가장 단순하고 쉽게 페이지 제작을 할 수 있도록 도와준다.

◎ jQuery 기능
* 문서 객체 모델과 관련된 처리를 쉽게 구현
* 일관된 이벤트 연결을 쉽게 구현
* 시각적 효과를 쉽게 구현
* aJax 애플리케이션을 쉽게 개발

◎ jQuery 특징
* jQuery는 자바스크립트 사용을 더욱 쉽게 하는게 목적이다.
* 사전 지식으로 HTML, CSS, JavaScript를 알아야 한다.
* jQuery는 적게 쓰고 큰 효과를 내도록 하는 경량화된 자바스크립트 라이브러리이다.
* 자바 스크립트를 더욱 쉽게 가볍게 만들어준다.
* Ajax나 DOM등을 다룰 때 복잡한 일들을 jQUERY 사용으로 단순화된다.
* 다양한 모든 브라우저를 가만해서 jQuery 제공한다.
